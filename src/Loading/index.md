# Loading

## 介绍

基础的 Loading 组件，支持自定义中心图片，文字提示，遮罩层等

## APi

| 属性   | 说明             | 类型    | 默认值    |
| ------ | ---------------- | ------- | --------- |
| url    | 图片地址         | string  | 曼城 logo |
| text   | 文字提示         | string  | 加载中... |
| isShow | 是否展示 Laoding | boolean | -         |
| isMask | 是否显示遮罩     | boolean | -         |
